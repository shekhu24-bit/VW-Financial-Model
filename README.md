# Volkswagen AG — Company Valuation & Financial Modeling

## Project Overview
Comprehensive financial model for Volkswagen AG built as part of MBA Finance coursework.
Objective: Estimate intrinsic value and determine if VW stock is over/undervalued.

## Model Structure (13 Sheets)
| # | Sheet | Description |
|---|-------|-------------|
| 01 | Cover | Professional cover page |
| 02 | Assumptions | WACC, growth rates, margin inputs |
| 03 | Historical P&L | Income Statement FY2021–FY2025 |
| 04 | Ratio Analysis | 15+ ratios + DuPont decomposition |
| 05 | Revenue Forecast | Bear / Base / Bull scenarios |
| 06 | Projected P&L | Forecasted financials 2026E–2030E |
| 07 | FCF Build | Free Cash Flow to Firm (FCFF) |
| 08 | WACC | Cost of Capital via CAPM |
| 09 | DCF Valuation | Enterprise Value → Intrinsic Price |
| 10 | Comps | 8 peer companies, trading multiples |
| 11 | Sensitivity | 2D WACC × Terminal Growth Rate |
| 12 | Football Field | Valuation range summary chart |
| 13 | KPI Dashboard | Executive summary + charts |

## Methodology
- **DCF Valuation** — FCFF discounted at WACC 5.32%
- **Trading Comparables** — EV/EBITDA, EV/Revenue, P/E vs 8 global peers
- **Sensitivity Analysis** — WACC 4%–9% × TGR 1%–3.5%
- **Scenario Analysis** — Bear / Base / Bull revenue projections
